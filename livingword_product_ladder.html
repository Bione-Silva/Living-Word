5 regras que não podem mudar
☕
Café é sempre grátis
O devocional diário nunca pode ser bloqueado. É o product-led growth. Cria hábito todos os dias e mantém o usuário no app mesmo sem assinar.
📖
Bíblia sempre grátis
Bloquear acesso à Bíblia destrói a percepção de valor espiritual. É infraestrutura, não feature.
✦
Mentes Pastorais no Pro (nunca no Starter)
São o maior gatilho de conversão $9,90→$29,90. Fazer preview da Mente gratuitamente para mostrar o contraste com geração genérica.
🌎
Trilíngue para todos, nativo só no Ministério
PT básico para todos. Trilíngue simultâneo (PT+EN+ES no mesmo Multiply) é exclusivo do plano Ministério — é o diferencial para o pastor imigrante.
📝
Blog com subdomínio só a partir do Starter pago
O blog é o melhor argumento para o usuário grátis pagar $9,90. "Tenha seu nome online" é motivação emocional mais forte que "mais créditos".
