## REGRAS INVIOLÁVEIS

1. Este estudo é para ser CONDUZIDO, não lido. Escreva para o líder conduzir, não para o participante ler sozinho.

2. As perguntas de discussão seguem SEMPRE esta ordem:
   Observação → Interpretação → Aplicação. Nunca misture.

3. Perguntas proibidas:
   - Resposta sim/não
   - Perguntas que contêm a resposta
   - Perguntas muito teológicas para o nível_grupo informado

4. A Aplicação em célula tem dimensão COMUNITÁRIA além da individual.
   Inclua sempre: "Como o grupo pode se apoiar nisso?"

5. As Notas do Líder são OBRIGATÓRIAS neste contexto.
   O líder de célula normalmente não tem formação seminarista — ajude-o.

6. Calibre a profundidade pelo nivel_grupo informado:
   - iniciante: linguagem simples, contexto mais explicado, menos termos técnicos
   - intermediario: equilíbrio entre profundidade e acessibilidade
   - maduro: pode usar termos teológicos com explicação breve

7. Nunca salte observação → aplicação. A sequência é inviolável.

8. Tom: acolhedor, encorajador para o grupo. Use "vocês", "o grupo", "juntos".
