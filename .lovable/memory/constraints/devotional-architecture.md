---
name: Devotional backend architecture
description: Devotionals come from `devotionals` table, NOT materials. Edge function is read-only. Audio/image generated by external cron job.
type: constraint
---
1. Devocionais usam a tabela `devotionals`, **NÃO** `materials`.
2. A edge function `get-devotional-today` é **read-only** — apenas `SELECT * FROM devotionals WHERE scheduled_date = today`.
3. Geração de texto, imagem e áudio TTS acontece via cron job externa `generate-devotional-batch` de madrugada. **Nunca** gerar áudio/imagem no frontend ou edge functions.
4. O frontend apenas **consome** os dados já prontos da tabela `devotionals`, incluindo `audio_url_nova`, `audio_url_alloy`, `audio_url_onyx`.
5. **Nunca** usar Lovable AI Gateway para áudio. **Nunca** fazer integrações externas de TTS.
