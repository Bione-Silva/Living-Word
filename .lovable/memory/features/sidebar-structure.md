Estrutura da sidebar desktop e mobile hamburger:

**FERRAMENTAS (topo)**
- Dashboard
- Criar (collapsible) — Estudo Bíblico, Blog & Artigos, Títulos, Metáforas, Ilustrações, Modernizador, Redator Universal, YouTube→Blog, Reels, Legendas, Newsletter, Avisos, Quiz, Poesia, Tradução
- Pesquisar (collapsible) — Tópicos, Versículos, Contexto Histórico, Citações, Texto Original, Lexical
- Bíblia

**RECURSOS (separator + label)**
- 🧠 Mentes Brilhantes (Premium)
- Sermões (rota /sermoes — promovido de "Criar")
- Modo Púlpito (rota /pulpito — NOVO badge) — lista sermões salvos, ao clicar abre /sermoes?materialId=X&pulpito=1
- Biblioteca
- Calendário
- Estúdio Social (rota /social-studio — promovido de "Criar")

**Bottom nav mobile (6 itens):** Início, Criar, Bíblia, Mentes, Sermões, Conta.

**Mobile hamburger menu inclui:** Início, Criar, Sermões, Modo Púlpito, Estúdio Social, Bíblia, Mentes, Ferramentas + bloco Conta.

Sermões e Social Studio NÃO aparecem mais dentro do grupo "Criar" da sidebar — moveram para Recursos para ganhar visibilidade.
